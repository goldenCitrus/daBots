# ***Conher and discord will prevail***

Conher is a simple discord text bot with many features including
##***Ratio***
        This has a 1.3% (1/75) chance of responding to a message sent within permanent 
channels with the simple phrase of “ratio”
	### ***Commands***
    ***Leaderboard***
		This is a command that any user can type that will display the top 10 most ratioed players within a given server
##***I Hardly Know Her***
This features checks a message and if it contains a word that ends with -er then Conher will simply response with “--er, i hardly know her”
    	## Fish detection
	  This checks every message that is sent and if a message includes a type of fish it will respond with a gif of Joel the spinning fish
                                                    https://tenor.com/view/fish-spin-sha-gif-26863370
##***Culling Games***
	 The Culling Games is a feature that is based on a Jujutsu Kaisen arc with the same name, if a user decides to join the game with the given command /join, while they communicate within their server they will collect points every time they get ratioed by Conher. Afterwards they can spend the default 5 points (This number could be change) to add a new rule that the players have to follow

    ###Rules
	The base rules are as follows:
        1. To Become a Player you must type /join into chat
        2. Points are earned when a Player gets Ratioed
        3. One Player could add a new rule using 5 Points
	Suggested rules:
		4. Players that break a rule will gain a penalty point for each rule broken
		5. ***A win condition rule to end the game***
		6. ***A punishment rule for breaking any of the previous rules***

	Each game will be different and will result in completely different results

	Enforcement of these rules are done by two methods, 
        1.A user adding to the code base and placing in checks to ensure that all players follow the rules
        2.A good faith agreement between the player to follow them and to call each other out if any one the added rules are broken

        Both of these methods have their pros and cons but the best results I would suggest doing a mix of both.

    ### Commands 
    ***current_rules***
        This responds this the current rules of the game
    ***view_points***
        Shows a leaderboard of everyone is has points
    ***add_rules***
        This commands allows user to add to the rules using up thier points
    ***join_the_game***
